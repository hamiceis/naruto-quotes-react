# Naruto Quotes
Naruto quotes generator 

<h3>technologies</h3>

<li> ReactJS </li>
<li> Typescript </li>
<li> Tailwindcss </li>

# Install
Clone this repository and install it dependencies with this command:
```sh
$ npm install
```

# Running
Run the application with `npm run dev` command, it will start the app on [localhost:3000](http://localhost:3000):
```sh
$ npm start
```
